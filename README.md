# DeanoBotsteano

Random fun physics facts with a twist

## Changing Token
Edit the `.env` file to add your discord token

## Running the code
Install the dependencies with
```
> npm install
```

Run the bot with
```
> npm run start
```

OR

```
> npm run build
> node build/index.js
```
